# Java Calculator

A simple calculator built using Java.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Technologies Used
- Java
- Scanner Class
- Switch Case

## How to Run

Compile:
javac Calculator.java

Run:
java Calculator

## Example Output
Enter first number: 10
Enter operator: +
Enter second number: 20
Result = 30
